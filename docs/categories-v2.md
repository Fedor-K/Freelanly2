# Freelanly Categories v2.0

На основе анализа Upwork, Fiverr, FlexJobs.

## Принципы отбора

1. **Только digital-first профессии** — работа выполняется на компьютере
2. **Remote-совместимые** — не требуют физического присутствия
3. **Востребованные на фриланс-рынке** — есть спрос на платформах
4. **Без лицензионных требований** — не требуют локальных сертификатов (CPA, адвокат)
